```

Ctrl+Alt+3 - Устав (2/3 часть) [ 3 Устава рандомно ]
Ctrl+Alt+4 - Устав (3/3 часть) [ 3 Сокращения рандомно ]
Ctrl+Alt+6 - Клятва [речь "Вы дали клятву гиппократу" ]
Ctrl+Alt+7 - Поручения для интернов [ 3 поручения рандомно ]
Ctrl+Alt+9 - Практика для парамедиков (1/2 частей) [ РП отыгровка с манекеном + речь что у него перелом ]
```
